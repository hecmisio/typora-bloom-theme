关于初心：在合适的光线里，慢慢生长

这个主题诞生于创作者成为父亲之后。开始更加在意时间、陪伴，以及那些会被反复翻阅的文字。我们相信，值得被记录的内容，也值得一个安静的、不随波逐流的界面。

设计理念：一天的光

Bloom 的设计灵感来自「一天的光」。两套主题在排版比例上保持绝对一致，只改变某种“光感”。

  • 🌤 Bloom Light (清晨) —— 清晰、柔和、低对比，像清晨洒入书房的第一缕阳光，适合白天阅读。
  • 🌙 Bloom Dark (夜晚) —— 克制、沉稳、护眼，像深夜台灯下的宁静，适合长时间专注思考。

设计原则

⌨️ 文字优先

所有设计都为内容让路。主题不抢表达，只负责让文字被更好地阅读。

⏳ 长期可用

不追求第一眼的惊艳，而追求在数月、数年后打开，依然感到舒适。追求耐看，而非流行。

🧘 情绪稳定

不追逐高对比，不制造感官刺激。让界面在白天与夜晚都保持平和的情绪输出。

🍃 极简克制

装饰性的元素被谨慎剥离。不是因为不能多，而是因为不需要多。

安装方式

  1. 打开 Typora。
  2. 进入：偏好设置 → 外观 → 打开主题文件夹。
  3. 复制主题文件：bloom-light.css、bloom-dark.css、bloom-forest.css、bloom-sea.css（如有 bloom/ 一并复制）。
  4. 回到 Typora：主题 → 选择 Bloom Light / Bloom Dark / Bloom Forest / Bloom Sea。

从 Releases 下载（推荐）

  1. 进入 GitHub Releases 页面。
  2. 下载最新的 Bloom-theme.zip。
  3. 解压后将文件复制到 Typora 主题文件夹。
